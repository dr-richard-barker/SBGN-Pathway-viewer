# ASGSR manuscript (in progress)

Home for the manuscript targeting the ASGSR journal **_Gravitational and Space
Research_**, built from the [OSDR Plant Pathway Atlas](../intro.md) in this repo.

**Working title:** *A FAIR, pathway-level atlas of the plant spaceflight
transcriptome: projecting NASA OSDR RNA-seq onto KEGG maps.*

## Planned structure
1. Introduction — plant spaceflight transcriptomics; the need for pathway-level,
   FAIR, reproducible synthesis across studies.
2. Methods — OSDR discovery → DE slicing → KEGG projection (the pipeline in
   `tools/`); the interactive SBGN/KEGG viewer.
3. Results — cross-study pathway matrix + heatmap; conserved vs study-specific
   responses (photosynthesis, cell wall/phenylpropanoid, hormone signalling,
   oxidative stress, circadian clock).
4. Discussion — biology, limitations, and the FAIR/reusable tooling.
5. Data & code availability — repo, Zenodo DOI, OSDR accessions.
